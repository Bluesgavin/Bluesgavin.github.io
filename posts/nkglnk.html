在直接围绕太阳运行的天体中，冥王星体积排名第9，曾与其他8个行星合称为“九大行星”。国际天文联合会（IAU）于2006年正式定义行星概念，将冥王星排除出行星行列，重新划为矮行星。曾经的“九大行星”重新变为“八大行星”。


《自然通讯》杂志发表的一项研究称，冥王星上存在巨大冰火山（蓝色区域） 图/NASA

我把自己的阅读习惯叫作冥王星计划。

在行星科学史上，如今太阳系的八大行星，都没有冥王星这样的际遇：真相难窥，多次新发现不断修正旧认知，导致身份改换，引发多种争议。每次人类对星空的探索，都是对自己定义的一次突破，我认为阅读也应如此。

下面是计划进行以来，我读过的书籍:

书名图片信息书评我的评分
大败局image.png吴晓波 / 浙江人民出版社 / 2001-0190年代是中国刚起步的时期，各类行业百花齐放。尽管不同时期的市场环境，投资环境都不一样，可即使在现在看来企业倒下的原因，却是出奇的相同。一个企业的兴衰，与其说是时代的浪潮，不如说是取决于掌舵者对浪潮的态度。⭐⭐⭐⭐⭐
大败局Ⅱ image.png吴晓波 / 浙江人民出版社 / 2011-5第二部案例集中在21世纪之后，我们可以看到很多在国退民进浪潮中的企业故事，以及金融和房地产大热的兴起。经济的进一步开放与资本在企业中的影响力逐渐增大，大大加速了企业的兴衰，也标志着中国经济进入了一个新的时代。⭐⭐⭐⭐⭐
关键对话 : 如何高效能沟通image.png（美）科里·帕特森（Kerry Patterson）、约瑟夫·格雷尼（Joseph Grenny）、罗恩·麦克米兰（Ron McMillan） / 毕崇毅 / 机械工业出版社这是一本非常棒的书，讲述了很多有用的沟通技巧。相信每个人都能从书中有所收获，反复阅读甚至可以人生。⭐⭐⭐⭐⭐
小岛经济学 : 鱼、美元和经济的故事image.png[美] 彼得·希夫、[美] 安德鲁·希夫 / 胡晓姣、吕靖纬、陈志超 / 中信出版社我一直都认为，经济学应该是所有非主修经济学的人的第二学科首选。它不一定会增加我们的财富，但可以让我们在生活中变得更加理性。而小岛经济学从一个非常通俗的故事，为我们阐述了经济的发展，用另一种方式感受经济的魅力。⭐⭐⭐⭐⭐
分布式服务架构：原理、设计与实战image.png李艳鹏、杨彪 / 电子工业出版社 / 2017-8工作需要看的这本书，偏工具类的书，实战时可以查阅。可是对于分布式从理论上讲的部分很少，后面的章节也很水，光docker也讲了很久。收获不大。⭐⭐
深入浅出Node.jsimage.png朴灵 / 人民邮电出版社 / 2013-12-1相信所有的前端或多或少都会用到nodejs，但由于真正用nodejs写服务的岗位并不多，所以很多人对node的认识只停留在api的使用。这本书可以很好的补充这个短板，书中把很多node的概念都讲得非常细，深入浅出，我认为可以选为每个前端都必读的作品之一。⭐⭐⭐⭐⭐
穷查理宝典 : 查理·芒格的智慧箴言录image.png[美]彼得·考夫曼 (编) / 李继宏 / 上海人民出版社查理是一个理性得让人可怕的人。他强大的逻辑能力，以及崇尚的跨学科论，可以给人更多的角度思考问题。书本本身内容大部分为一些收录，会相对比较难啃，不过在阅读过程中能体验到与智者交流的感觉。⭐⭐⭐⭐
人间失格image.png太宰治 / 许时嘉 / 吉林出版集团我没有能力去发现这部作品的文学价值。只能单单从内容上思考，究竟什么人会去看这篇小说?我一直以为小说的形式可以是千奇百怪的，但他的核心思考应该是积极的，值得被推荐的。可是在这部如此颓废的作品，我只看到了绝望。由此并不推荐把他当成休闲的阅读。⭐⭐⭐
漫画算法：小灰的算法之旅image.png魏梦舒 / 电子工业出版社 / 2019-5算法的解析很形象，通俗易懂。加入的场景也很贴切，是一部值得推荐的算法入门作品。⭐⭐⭐⭐⭐
图解HTTPimage.png[日] 上野宣 / 于均良 / 人民邮电出版社清晰地讲解了互联网的原理结构和运行流程。末章安全问题的讲解更是比我以往网上看到所有资料都要具体，详细很多。相信这本书对所有web从业者都会有收获。⭐⭐⭐⭐⭐
一往无前 : 雷军亲述小米热血10年image.png范海涛 / 中信出版集团 / 2020-8在我的印象中小米一直以来都像是一个在模仿的公司。但感谢这本书让我有一次，从零开始陪伴小米成长。我看到的是一家对梦想有着超乎寻常执着的公司。小米不仅改变了中国的工业玩法，并用自己的创新把“中国模式”带上国际。⭐⭐⭐⭐⭐
JavaScript设计模式与开发实践image.png曾探 / 人民邮电出版社 / 2015-5相信对于没有学过其他语言的“纯前端”来说，很难可以真正了解到“面向对象”的概念。而很多的设计模式又是基于面向对象设计的。本书从JavaScript中面向对象的利用为入口，用js介绍了几种常见设计模式。并提出了代码优化的思路。整体逻辑连贯通畅，我认为本书对所有想要进阶的前端开发者都会有很大的帮助。⭐⭐⭐⭐⭐
黑客与画家image.png[美] Paul Graham / 阮一峰 / 人民邮电出版社相信每个人从小到大，身边或多或少都会存在几个“书呆子”。其实我心理很羡慕他们的能力。但往往又会站到大众的一边来嘲笑他们，现在想起来也算是一种可悲。感谢本书让一个“书呆子”给我们表达了那么多的观点。无论是对社会的看法，还是从专业上对程序的思考都是十分深刻的。对比之下很容易可以发现，很多时间我们做不到，只是我们根本没有真正的深入思考罢了。⭐⭐⭐⭐⭐
程序员修炼之道 : 从小工到专家image.pngAndrew Hunt、David Thomas / 马维达 / 电子工业出版社这本书更多的是从“程序员”这个职位出发，详细讲述了从需求，开发，测试到交付整个流程的具体细节。区别于具体开发技能等“硬实力”的讲解，这本书带来的更多是岗位上下游相关的一些“软实力”分享。其实跟所有的岗位一样，如果想要提高自身的产出，除了简单地完成自己的工作以外，还要懂得在自己的业务线上发挥额外的作用。⭐⭐⭐⭐
UNIX传奇 : 历史与回忆image.png[美] Brian W. Kernighan / 韩磊 / 人民邮电出版社UNIX的诞生不仅仅是操作系统历史的一个里程碑，更是科学与应用的一次交棒。 在这本书里我第一次如此接近地了解到，原来我们所用的工具都是一次一次的科学研究的产物。一个开发者除了完成日常的业务开发之余，应该还要花时间在研究思考上。⭐⭐⭐⭐⭐
高效能人士的七个习惯 image.png[美] 史蒂芬·柯维 / 高新勇、王亦兵、葛雪蕾 / 中国青年出版社曾经有朋友问我，有没有一本书是让我读了之后感到相见恨晚的？我想就是这本了。思维是很主观的东西，但人类社会中大部分的思路都是有共性的。我认为这本书很好地总结了人类社会的思考，成长的路线。为阅读者提供了一个具有高参考价值的人生指导，相信每个读过这本作品的人都会有收获。⭐⭐⭐⭐⭐
Vue.js设计与实现image.png霍春阳 / 人民邮电出版社这一定是2022年前端领域最好的作品，深入浅出地从设计到实现讲述了VUE。对开发者对渲染框架的理解有很大的帮助，个人觉得也是近几年来业内最好的书籍之一。⭐⭐⭐⭐⭐
破局 : 全面提升你的竞争力image.png月夜生凉 / 四川文艺出版社这是一本可以让你在迷茫中找到角度的作品。可以看出作者有着大量的阅读知识，以及对世界有独特的看法。但其实想要解析社会的运作，就很难避免对过去经典著作的参考。这也使得这部作品在某些层面上很难脱颖而出。⭐⭐⭐
杀死一只知更鸟image.png[美] 哈珀·李 / 李育超 / 译林出版社长大之后我开始明白，我们在人生每个不同的阶段都有需要面对的问题，需要追求的内容。但纵观人生，总有一些东西是我们应该用一生追逐的，那就是善良和勇敢。⭐⭐⭐⭐⭐
经济学原理 : 微观经济学分册image.png[美] N.格里高利·曼昆 / 梁小民、梁砾 / 北京大学出版社这虽然是一本教材，但通俗易懂并不会让人觉得无聊。经济学是一门涉猎很广的科学，可以尝试解释我们生活中大部分场景，是所有社会人提升软技能的最佳选择。⭐⭐⭐⭐⭐
重构 : 改善既有代码的设计image.png[美] Martin Fowler / 熊节、林从羽 / 人民邮电出版社 很幸运这本书的二版改用了JavaScript来写，对我来说简直是福音。作为一个开发者，我们有时候太过关注于应用级别的内容，以至于忽略掉最底层的开发。本书带来了很多纯粹的开发层面的知识，干货满满，是每位开发者必读的作品。⭐⭐⭐⭐⭐
